    Projetos de Ciência de Dados

    Projeto de Classificação quanto a propensão de Vendas de uma Seguradora.

    Foi usado Python, Estatística e técnicas supervisionadas de Machine Learning para classificar um grupo de clientes sobre a aceitação dos mesmos a um novo produto de seguro de carros, usando como resposta um questionário de interesse. Objetivando classificar em uma lista raquiando os clientes mais propensos a compra, através de técnicas de Learning to Rank. Esta solução ao ser implementada obtém uma performance de assertividade de 300% a mais do que se fosse entrar em contato de forma desordenada contato com 20 mil clientes de uma lista de 127 mil disponíveis que a equipe de marketing entrara em contato.

  • Python, Pandas, Matplotlib e Seaborn.
  • Jupyter Notebook;
  • Git e Github;
  • Cloud Heroku.
  • Flask e python API's.
  • AWS Cloud(Postgres, SQlite).

    Projeto de Previsão de Vendas de uma Rede de Farmácias.

    Usando Python e técnicas de regressão de Machine Learning foi feita a previsão de desempenho das vendas de uma rede de farmácia para as próximas 6 semanas melhorando a métrica de previsão de vendas. O modelo ficou hospedado em plataforma cloud e foi desenvolvido um bot para ter acesso via celular. Como resultados, conseguiu-se fazer uma previsão com erro médio maximo de 9,7% na previsão, com isto é possível que a rede consiga trabalhar com cenários negativos e positivos dentro desta variação de vendas.

O principal objetivo era gerar bons insghts sobre oportunidades lucrativas no ramo imobiliário através da Exploração e analise dos dados.
